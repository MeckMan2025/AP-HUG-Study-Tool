# AP Human Geography Study Tool

An interactive, browser-based study tool for the AP Human Geography (APHUG) exam. Open `index.html` in any modern browser — no install, no server, no dependencies.

## Features

- **Interactive Maps** — Explore and quiz yourself on regions using Leaflet.js maps (US States, Europe, Latin America, Sub-Saharan Africa, SW Asia & N Africa, South & SE Asia, and World Countries)
- **Map Quiz Mode** — Click the correct country/state on the map when prompted; tracks your score and streak
- **Reading Quizzes** — Multiple-choice questions organized by unit and topic (Units 5 & 6 currently have topic-level quizzes)
- **Redo System** — Missed questions are saved to your Redo list so you can drill your weak spots
- **Score Tracking** — See your score and streak in real time as you quiz
- **Mobile Friendly** — Responsive layout works on phones and tablets
- **No Backend** — Everything runs client-side; progress is stored in your browser's localStorage

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/MeckMan2025/AP-HUG-Study-Tool.git
   ```
2. Open `index.html` in your browser.

That's it — no build step, no dependencies.

## How to Use

1. **Pick a unit** from the sidebar (click the menu button at top-left).
2. **Explore mode** — hover over map regions to see names and details.
3. **Quiz mode** — click "Quiz" in the header, then click the correct region when prompted.
4. **Reading quizzes** — select a reading quiz topic from the sidebar to answer multiple-choice questions.
5. **Redo** — review and retry questions you got wrong.
